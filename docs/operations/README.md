## About This System

MycoERP is a complete farm management system designed for commercial mushroom cultivation operations. It tracks every batch from petri dish to packaged product, assigns and monitors tasks for workers, provides QR-based physical traceability, monitors environmental conditions via IoT sensors, and generates production analytics.

## Supported Languages

The application is available in English and Bengali. You can switch languages at any time using the language selector in the top header bar.

## User Roles

The system has six roles, each with different levels of access:

- **Admin** -- Full system access, manages users, devices, and permissions
- **Farm Manager** -- Oversees production, approves tasks, manages batches and templates
- **Lab Worker** -- Manages agar cultures, liquid cultures, and grain spawn batches
- **Production Worker** -- Manages substrate, fruiting blocks, and production workflows
- **Harvest Worker** -- Records harvests, completes assigned tasks, reports contamination
- **Viewer** -- Read-only access to all operational data

For detailed role permissions, see the [Administration Guide](./admin-guide.md).
